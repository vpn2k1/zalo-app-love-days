# Trailing Commas

Use trailing commas in multiline:

- object literals
- arrays
- function parameters and arguments
- TypeScript type members where the surrounding style uses commas
- JSX props split across lines when formatted that way by tooling

Do not fight the formatter. If nearby code has a consistent format, match it.

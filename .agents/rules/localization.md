# Localization

The app is currently Vietnamese-first and does not use an i18n framework.

- Keep user-facing copy in Vietnamese unless the user asks for another language.
- Keep copy natural, concise, and appropriate for couples using a mobile Mini App.
- Do not translate product names, service names, env var names, or code identifiers.
- Avoid scattering repeated long strings; extract repeated copy only when reuse is real.
- If adding formal i18n, ask first because it affects app structure and copy organization.
